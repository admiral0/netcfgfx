Dependencies

Project needs:
Compile time:

  • qt-dev
  • g++
  • iwlib.h
  • cmake

Runtime:
  • qt
  • iwlib
  • netcfg
  • sudo(configured correctly)

Install

  • mkdir build
  • cd build
  • cmake ..
  • make
